# Tombola Natalizia Online

Un'applicazione web per giocare alla tombola online con amici e familiari, con un tema natalizio.

## Caratteristiche

- **Pannello Amministratore**: Crea stanze di gioco, estrai numeri, gestisci i giocatori
- **Area Giocatore**: Unisciti a stanze con un codice, gioca con cartelle personali
- **Tema Natalizio**: Interfaccia festosa con effetto neve
- **Responsive**: Funziona su desktop, tablet e smartphone
